# Tech Challenge 
# 📊 E-commerce Data Analysis – Olist Dataset

## Introdução

O crescimento do comércio eletrônico gerou grandes volumes de dados relacionados a pedidos, clientes, pagamentos e entregas. A análise desses dados permite compreender padrões de comportamento dos consumidores, identificar gargalos operacionais e apoiar decisões estratégicas baseadas em dados.

Neste projeto foi realizada uma Análise Exploratória de Dados (EDA) utilizando o dataset público de e-commerce da Olist. O objetivo é investigar padrões de compra, avaliar a experiência do cliente e identificar insights relevantes para o negócio.

---

## Objetivo do Projeto

O objetivo deste trabalho é explorar e analisar dados de uma plataforma de e-commerce para identificar padrões e gerar insights que possam apoiar decisões estratégicas.

Entre os principais pontos analisados estão:

- evolução do volume de pedidos ao longo do tempo
- métodos de pagamento mais utilizados
- distribuição das avaliações dos clientes
- impacto de atrasos na entrega na satisfação do cliente
- categorias de produtos com maior volume de vendas
- distribuição geográfica dos clientes
- análise do ticket médio das compras

---

## Dataset Utilizado

Foi utilizado o dataset público:

**Brazilian E-Commerce Public Dataset by Olist**

Este conjunto de dados contém informações sobre pedidos realizados em um marketplace brasileiro, incluindo dados de clientes, produtos, pagamentos, vendedores e avaliações.

O dataset permite analisar diferentes aspectos do funcionamento de uma plataforma de e-commerce, como comportamento de compra, desempenho logístico e satisfação dos clientes.

## Principais Análises Realizadas

Durante o projeto foram realizadas diversas análises exploratórias, incluindo:

- análise da evolução do número de pedidos ao longo do tempo
- identificação dos métodos de pagamento mais utilizados
- análise da distribuição das avaliações dos clientes
- investigação de atrasos nas entregas
- relação entre atraso de entrega e avaliação do cliente
- identificação das categorias de produtos mais vendidas
- análise da distribuição geográfica dos clientes
- cálculo e análise do ticket médio dos pedidos
- análise de correlação entre variáveis

Essas análises permitiram compreender melhor o funcionamento da plataforma e identificar padrões importantes no comportamento dos clientes.

---

## Principais Insights

A análise exploratória permitiu identificar alguns insights relevantes:

- houve crescimento consistente no volume de pedidos ao longo do tempo
- o cartão de crédito é o principal método de pagamento utilizado pelos clientes
- a maioria das avaliações dos clientes é positiva
- atrasos nas entregas tendem a impactar negativamente a avaliação dos clientes
- algumas categorias de produtos concentram maior volume de vendas
- existe concentração de clientes em determinados estados do Brasil

Esses insights demonstram como a análise de dados pode contribuir para compreender o comportamento dos consumidores e apoiar decisões estratégicas.

---

## Recomendações de Negócio

Com base nos resultados obtidos, algumas recomendações podem ser consideradas:

- investir em melhorias logísticas para reduzir atrasos nas entregas
- priorizar estabilidade e segurança em métodos de pagamento mais utilizados
- desenvolver estratégias de marketing focadas nas categorias mais vendidas
- explorar oportunidades para aumentar o ticket médio dos pedidos
- utilizar dados geográficos para orientar estratégias regionais de expansão

Essas ações podem contribuir para melhorar a experiência do cliente e aumentar o desempenho do negócio.

## Conclusão

A análise exploratória de dados permitiu identificar padrões importantes no comportamento de compra dos clientes e no desempenho logístico da plataforma de e-commerce.

Os insights gerados demonstram como a utilização de dados pode apoiar a tomada de decisões estratégicas, contribuindo para melhorias na experiência do cliente, eficiência operacional e crescimento do negócio.

Este projeto reforça a importância da análise de dados como ferramenta fundamental para organizações que atuam em ambientes digitais e altamente competitivos.
